  1. Q: What element is used to create images? Name its attributes.

    Ans: < img > is the tag used to create an image in HTML. The src attribute holds the image location, the alt attribute holds the name of an image if it is not visible on the web page, the width and height attributes dictate how big the image will be, and the title attribute shows the name of the image when the cursor is hovered over the image ont he web page.
