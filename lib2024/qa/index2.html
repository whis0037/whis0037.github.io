  2. Q: Define version control and give an example

    Ans: Version control is the saving of different versions of a file, folder or document so that you can revert back to it if you need to. Microsoft Word does this so that if the computer crashes you can open the last saved version and not lose all the work you did.

  3. Q: What tag is used to create the title of the web page? Give an example

    Ans: < title > "title of the web page" < /title >

  4. Q: What tag is used to create a paragraph in HTML?

    Ans: < p > is the tag used to create a paragraph in HTML

  5. Q: Identify the odd one from these tags: <span >, <p >,<a >,<i >

    Ans: < i > is the odd one out as it alters the text so that it is in italics. The other tags are structural elements.

  6. Q: What are the tags required to create a navigation list in HTML? Give one example with two navigation links.

    Ans: The < nav > tag is required to create the container for the navigation list. < ul > and < li > are the tags needed to create the list itself. < a > tag creates a link to which a URL is added to take the user of the link to the website.
